# Sistema de RH em Python

Neste projeto, eu desenvolvi um programa em Python que simula um sistema simples de RH para calcular o salário final de um funcionário.

A proposta desse código foi trabalhar com entradas de dados, estruturas condicionais, laços de repetição e funções. O programa solicita informações como nome do funcionário, cargo, salário base, horas extras, faltas no mês e se houve bônus por desempenho.

Com base nesses dados, o sistema calcula:
- o valor das horas extras
- o desconto por faltas
- o bônus, de acordo com o cargo
- o salário bruto
- o salário final

Esse foi o meu primeiro projeto nesse genêro, então busquei deixar o código de uma forma mais organizada possível.

## Objetivo

Desenvolvi esse código para praticar e me aprimorar em Python como forma de me desafiar e tentar buscar soluções para problemas reais.
